# TouchenceSample
Touchence社が開発した，CUB-D-16D-SN20-CNを読み取るサンプルプログラムです．センサ等の詳細は，http://www.touchence.jp/cube/に記載されています．

## Overview
* POT-DEB-CNを通じたシリアル接続で，複数のセンサデータを読み取ることが出来ます．
* TCP/IP接続を通じて，クライアントにセンサデータを処理した結果を送信できます．

## Desription
* シリアル接続を通じて得られたセンサデータを処理し，クライアントに送信します．クライアントとの接続が確立されると自動的にデータを送信します．データの送信を停止もしくは開始したい時は，"Stop\n"もしくは"Start\n"を送信してください．
* 複数のPOT-DEB-CNにも対応するためのコードは書いていますが，まだテストできていません．
* シリアル接続や通信に失敗した時の再接続機能はまだ実装していないので，エラーが起きた場合にはプログラムを再起動してください．

## Requirement
このプログラムはVisual Studio 2015で作成されたので，ビルドする場合には同じ環境を利用するのが良いかと思います． 

## Usage
* Configフォルダに，各種設定が記載されたSettings.xmlがあります．
* "comport"にCOMポートの番号を，"id" と "calibrationfilename" には実際に利用するセンサにアサインされているIDと校正用ファイルを指定してください．
* 起動すると表示されるGUIに背景差分を取るためのボタンがあるので，それを押すことで背景差分用ファイルの値が更新されます．
* 現時点で対応しているセンサの種類（"sensortype"）はCUB-D-16D-SN20-CNのみです．
